# UU项目图片存储仓库

这是一个用于存储UU社交网站图片的GitHub仓库。

## 用途
- 存储用户上传的图片
- 通过jsDelivr CDN加速访问
- 提供永久免费的图片存储服务

## 文件夹结构
- `images/` - 主要图片存储目录
  - `blacklist/` - 黑名单证据图片
  - `avatars/` - 用户头像图片
  - `posts/` - 帖子内容图片

## 访问方式
图片可以通过以下URL格式访问：
```
https://cdn.jsdelivr.net/gh/yoxoz/UU-images@main/images/文件名.jpg
```

## 注意事项
- 请勿上传敏感或违法内容
- 图片会自动通过CDN加速
- 仓库为公开仓库，请勿存储私人信息
